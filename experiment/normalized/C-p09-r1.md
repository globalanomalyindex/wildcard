## angles

- Air traffic controllers metering demand with committed slot times instead of blocking → enterprise tenants defaulting to top-of-hour exports because of lack of visibility → expose a lightweight schedule negotiation endpoint where tenants receive committed future windows within per-window cost budgets, spreading load without hard rejection or SLA conversations.

- Epidemiologists reconstructing transmission chains from partial data retroactively → tenant ID threading being inconsistent across the system → instrument edges that are controllable (load balancer, ClickHouse query log, export job trigger) and build a probabilistic attribution layer correlating timing, client IP ranges, query fingerprints, and tenant fragments to achieve 80% attribution confidence and enforce soft limits without waiting for threading refactor.

- Utilities shifting peak demand via interruptible tariff contracts paying customers to move load → enterprise customers experiencing SLA violations from hard rate limiting → offer scheduled export credits where customers commit to staggered export windows in exchange for concrete incentives (price concession, higher API quota outside peaks, guaranteed p95 target instead of p99), reframing the SLA problem into joint optimization.

- Acequia systems allocating water rights by time slot with enforcement via structural gates that open and close on schedule → need for real-time consumption policing removed by making the schedule itself the mechanism → implement pre-execution query budgeting at the ClickHouse layer using resource groups or query complexity limits tied to per-tenant tokens attached at ingress, moving enforcement to the query planner itself and making it survive deploys.
