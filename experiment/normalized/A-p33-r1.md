## angles
- Ground delay costs nothing; in-the-air vectoring costs everything. Pre-commitment in air traffic is expensive; the cheap intervention is resolving the imbalance before it materializes. -> Dispatching a van in response to a call commits it to an expensive trajectory, missing nearby calls it could have taken cheaply. -> Pre-position vans in geographic sub-zones at start-of-day based on predicted demand, so they are already proximate when the call arrives.
- Sector delay can propagate as compression through downstream sectors; the failure mode is cascade, not just busy capacity. -> The five-day summer wait queue is compression cascade running through a system with no pre-season absorption mechanism. -> Implement pre-season preparation (seasonal hire, parts pre-stock, zone assignment) before June, not in response to June.
- You cannot work a demand-capacity balance against a number you don't trust. -> Without a reliable inventory count, every parts-demand match is a guess; capacity is distributed across nine vans plus the room, making the problem harder. -> Maintain a van-inventory manifest, verified and updated after every job, so you know which sector holds the capacitor the next call will need.
- Miles-in-trail is a spatial metering rule that prevents bolus congestion downstream. -> The operation has no spatial structure; each van is free to take any call anywhere, resulting in crisscross routing equivalent to aircraft converging on the same fix at once. -> Assign soft geographic zones as default home-sector assignments, with dispatcher override when local capacity is exhausted; the van nearest the call handles it by default.
- Collaborative decision-making only works when both sides can see each other's state. -> The dispatcher doesn't know van position, current job difficulty, remaining capacity, or which calls match skill level, and the van doesn't know the queue; neither side can negotiate well. -> Create a shared information structure (physical or digital whiteboard) showing van location, current job status, today's remaining capacity, and a flag for calls needing the senior tech.
