## angles
- Ecologists use indicator species to assess ecosystem health by monitoring a small set of signals rather than everything. The 27 actionless pages are noise; regulated payment-flow paths are the true indicators. Each alert should be traced backward to ask whether it has ever been the first signal of regulated-path failure. Alerts that cannot show that lineage can be demoted to dashboards. This reframes the project from "raise thresholds" to "build a verified indicator set," which compliance officers accept more readily as an argument about signal fidelity rather than laxity.

- Aviation's MEL (minimum equipment list) is pre-negotiated, documented, and regulator-approved before the crisis. The inverse problem here is MEL negotiation happening during the crisis, under duress and without documentation. Run a structured "alert MEL" exercise now, in a non-urgent window, where each alert is classified as either "regulated-path-critical" or "advisory." The output is a written, dated document that compliance can review. Because it is prospective and documented rather than reactive and informal, it tends to survive audits better than threshold tweaks.

- In burn medicine, aggressive removal of dead tissue (necrotic material) lets healthy tissue survive; leaving it in place does not protect the wound but infects it. The 27 actionless pages are necrotic tissue, actively degrading the on-call system by training engineers to treat all pages as probably meaningless. The reframe is that alert retention (not reduction) is a risk. This may be useful when talking to leadership: the proposal is not to lower the guard but to remove the dead material that is currently preventing it from functioning.

- Toyota's production system separates continuous monitoring (everything watched on dashboards) from the andon cord (a human-initiated stop-everything signal for an actual defect). Software on-call rotations wire everything directly to the andon cord, making it meaningless. Route batch-job CPU and disk alerts to a shared channel that is visible but silent overnight. Reserve true pages for alerts mapped in writing to a regulated path. Engineers then pull the cord themselves if they notice something in the channel that looks like a real failure. This does not require resolving the classification problem first; it just changes the default interruption level while the classification work proceeds.
