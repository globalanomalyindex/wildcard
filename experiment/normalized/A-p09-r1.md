## angles
- Platoon interference from independently timed schedulers -> p99 spike is a temporal concentration problem, not a volume problem -> offset scheduled export windows across the hour to spread arrivals rather than accepting synchronized bursts
- Green wave principle: phase offset between adjacent signals -> same infrastructure handles distributed arrivals instead of synchronized peaks -> stagger tenant A to :00, tenant B to :12, tenant C to :24 across the hour
- Spontaneous re-synchronization regenerates the problem as new tenants independently choose round-number defaults -> opinionated scheduler UI that offers staggered windows and makes top-of-hour a choice, not pre-fill -> builds the offset constraint into the default configuration so new tenants don't recreate the collision point
