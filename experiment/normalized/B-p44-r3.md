## angles
- Measurement is being placed at the wrong level of the system (compound metric combining bass-clef reading, rhythmic internalization, and finger pattern recall) -> when any component fails, the blockage is masked -> measure each component separately (e.g., "did I read that bass note before playing it, or did I guess").
- The diagnostic measurement needed is not absolute performance but the fit threshold between current ceiling and book difficulty (the gap) -> restarting from page 1 treats the question as "am I a beginner" when it should be "where does the book first exceed my current ceiling" -> open to the page where difficulty first exceeds current ceiling, not page 1.
- Skills have different consolidation floors and may not accumulate when exposure stays below the threshold for symbolic lookup (bass-clef fluency) -> 20 minutes distributed across everything may fail to cross the floor for any single component -> concentrating all 20 minutes on one component (bass clef alone, for a week) might cross the consolidation threshold that mixed sessions do not reach.
