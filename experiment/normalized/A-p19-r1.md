## angles
- Two incompatible measurement systems (mold units vs. piece units; host units vs. request units) cannot collapse into one without loss; the solution is a reliable conversion table baked into workflow practice rather than specs -> trial pricing and user expectation mismatch requires persistent translation inside the product from first session, converting host count into user's native unit throughout (running "at your current request volume, this maps to N hosts")
- Invisible accumulation at a critical threshold (pour-out timing in casting; host limit onset during onboarding) creates failure that appears only later, but damage is done when everything seems normal -> surfacing the hidden variable while time remains to act (making wall thickness visible during casting; making proximity to host limit a live signal during trial) converts cliff into ramp without changing tier structure
- Lagging indicators (fired piece quality; exit survey "felt tricked") reflect upstream degradation that happened earlier (mold porosity loss; trust infrastructure mismatch formed before host limit is hit) -> shift diagnostic focus to the earlier, less-visible step where expectation and reality diverge, not the visible late-stage expression
