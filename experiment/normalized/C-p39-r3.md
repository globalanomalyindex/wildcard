## angles
- Emergency-room triage tags (physical color-coded status attached to asset) -> returned machines lack real-time visibility between physical arrival and system update -> pilot a physical tag system at the wash bay (red/yellow/green on hooks) as source of truth instead of keyboard records
- Aviation checklist culture (two-person read-aloud ritual makes accountability collegial) -> solo mechanic walk-arounds lack social pressure to be thorough and to document consistently -> mount a laminated checklist at eye level on machine, have customer co-check-in by reading items aloud while mechanic confirms each line
- Supermarket shelf-gap detection (designed for visibility of absence rather than presence) -> high-value assets missing for weeks go unnoticed because system counts utilization, not non-movement -> designate numbered and painted parking stalls for largest assets; daily visual sweep makes empty stalls a signal rather than silence
- Package-sort hub wave scheduling (separate "received" event from "cleared" event in tracking system) -> Saturday 7-9 am return crunch creates false availability because returned machines are immediately marked rentable before inspection -> define two distinct system states ("returned, pending check" vs "available for rental") and enforce rule that nothing moves from returned to available without cleared tag
