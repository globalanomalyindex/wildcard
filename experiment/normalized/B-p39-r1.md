## angles
- A gauge block carrying a hidden offset from contaminated intake propagates the error invisibly downstream → the "available" machines not yet inspected are the same structural problem (logical state looks certified but material state is unknown) → represent "returned, uninspected" as a distinct third state so contaminated blocks don't sit in the same tray as clean ones
- Nominal spec rating is decoupled from present surface-state assessment; only current condition tells you readiness → inconsistent check-in photos mean the assessment either happens or doesn't at the one moment it matters (return intake) → a standardized surface-inspection protocol with a structured checklist run at every return, before the machine re-enters the certified pool, makes the assessment reproducible and disputable from evidence
- Certified and uncertified blocks must never share a tray, even if tracked mentally, because time pressure during high-volume intake makes mixing too likely → the Saturday surge with one loader and one forklift driver is where "I'll remember which haven't been checked" fails → route returns to a physically distinct zone before inspection clears them, so the system enforces the rule rather than memory
