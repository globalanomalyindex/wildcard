## angles
- In cuneiform administration, signs maintain identity through stable relational function despite enormous surface variation across media and tools. The load-bearing features (which if altered cause misreading) can be distinguished from free-to-vary surface features through systematic testing. -> The donor recognition survey likely contains data on which features of the old mark are doing the work, but has not yet isolated them. Before the $12k deployment, test which features drive recognition from retired teachers by showing them stripped-down variants (color only, shape only, texture only) to see where recognition drops. The rebrand may have preserved all load-bearing features and only changed tolerable surface variation, or it may have discarded something critical. -> Run a feature-isolation pass on existing survey data before committing budget.

- Provincial Ur III scribes worked under time pressure (clay dries fast), tool degradation (reed styli wear), and distributed training (no access to central scriptorium). The solution was not standardization (impossible under constraints) but a tolerance threshold: a minimum set of non-negotiable features that define identity, with everything else free to vary. -> Your chapters print locally with no shared asset server and $12k cannot build infrastructure to solve that constraint. The question is not "how do we achieve a single canonical mark everywhere" but "what minimum features must be preserved for the mark to function?" -> Produce a tolerance document defining three non-negotiable features and what can vary without harm. This costs almost nothing and can be emailed to every chapter printer tomorrow.

- In cuneiform scholarship, there is a productive distinction between the canonical reading (stable reference for publication) and the working variant corpus (where all provincial variants, adaptations, and scribe-specific features live for actual research). Insisting the canonical version is the only legitimate version causes damage. Both tiers are legitimate and serve different functions. -> You have a donated rebrand (canonical text) and a working variant corpus (nine versions, old mark, chapter adaptations). The implicit question "which one is the real logo" may be wrong. -> Use the canonical rebrand for external-facing contexts (grants, new donor outreach, national communications) and the variant corpus for local recognition contexts (chapters, retired teacher donors, existing workbooks). This is a stable institutional arrangement, not a transitional phase. The question becomes "where does each version belong and who decides"—a governance question cheaper to resolve than a production question.
