## angles
- Substrate-driven divergence often masks fewer source files than the surface count suggests; the same source file across different media (screen-printing, embroidery, different clay substrates) produces different-looking marks -> Nine logo versions may descend from only two or three actual source files -> Request the actual files (AI, EPS, PDF) from every chapter and map each version to its source; this 6–8 hour audit reveals whether the problem is a nine-remediation task or a one-file replacement plus redistribution, reshaping what budget should fund.
- Distributed archives need explicit authorization signals built into the artifacts themselves, or readers cannot distinguish authorized copies from forgeries or errors -> Chapter version entropy may stem from ambiguity about what is sanctioned, not from resistance to the rebrand; a chapter using an old file under uncertainty is rational -> Write a one-page "Two-Mark Policy" that explicitly authorizes both marks with defined domains (e.g., crayon-style for donor-facing print through a stated date; new mark for all digital and partner-facing materials), distribute it alongside two clean master files to a single named contact per chapter, formalizing what is already true and removing the ambiguity that drives entropy.
- The dangerous state is not mixed current states but mixed states with no documentation, because you cannot later tell convergence from undocumented new variants -> Without a record of which chapter is in which state and why, a year-later check cannot distinguish 70% convergence from three new unauthorized variants -> The chapter-audit spreadsheet from the first move doubles as a living convergence record; add "target mark" and "transition date" columns and update on each transition, giving you a board-meeting table showing "14 of 17 chapters authorized on new mark for digital; 17 of 17 on crayon mark for year-end print" and an evidence base for the rebrand team.
