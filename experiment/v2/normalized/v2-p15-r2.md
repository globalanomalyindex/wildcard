## angles
- Border adopters (chapters nominally using the new mark but printing from unofficial low-res sources) represent the actual risk zone, not clear holdouts -> these create undetected, uncontrolled variant production -> audit the chapters you believe have already adopted; ask each for the file they're printing from before spending rollout budget.
- Classroom workbooks are immovable spokes that cannot be recalled at realistic budget -> trying to unify the new mark across fixed printed material warps both systems and propagates failure -> formally define two zones (new print/digital/replaceable items get the new mark; classroom workbooks and existing distributed materials stay with the old mark, listed as permanent) and stop trying to unify them.
- The recognition link driving year-end giving was built on specific association with the crayon-style visual form, a different force vector than the strategic coherence the rebrand was designed against -> more rollout effort applies force from the wrong direction if the link is mark-dependent -> before allocating rollout budget, run a recognition test with 30-40 retired-teacher donors (15 questions mixing new mark, old mark, and variants) to determine whether the giving signal transfers to the new mark or is specific to the crayon visual elements.
