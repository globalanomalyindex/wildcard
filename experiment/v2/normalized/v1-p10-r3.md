## angles
- Figma library as wave description (full superposition of states) and React package as particle description (discrete, actualized implementations) are valid but mutually exclusive accounts -> The drift between them is not a failure but evidence they are being treated as one description -> Build an explicit conjugate pair protocol that specifies what is preserved, lost, and acknowledged when translating between them, rather than trying to unify the two systems.
- Whether wave or particle behavior appears depends on measurement timing -> In this context, the measurement cadence (when Figma gets compared to production) determines whether divergence is bounded or accumulates -> Specify "when does Figma get compared to production" as a measurement cadence question in the experimental design to catch divergence each sprint rather than at customer review.
- Every measurement collapse in quantum mechanics is a logged, deliberate event with recorded alternatives -> Each Figma detach is a collapse event that currently happens silently -> Log every detach with the specific variant, state, or prop that the React package couldn't express; sorted by frequency, this log becomes a measurement dataset and engineering prioritization backlog with tractable targets.
