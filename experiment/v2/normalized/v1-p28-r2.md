## angles
- Hidden dimensions of internal state (rights status) are not encoded in outputs and cannot be reconstructed from inspecting them alone -> archive failures stem from controllers making decisions on partial state -> build a state observer that reconstructs rights status from measurable signals (production date, performer name, contract archive, union agreements) rather than touching each image file
- Retrieval system has one dominant attractor (chronological search) that captures all query types into the same behavior -> all queries regardless of content resolve to manual date-index walking -> restructure the landscape by adding a second stable basin, e.g., a production-title index or costume-color tags on high-value images, to pull certain query classes into an alternative basin without touching the rest of the archive
- Archive's design is optimized for steady-state (fully tagged, fully cleared, full semantic index) but is operating in a long transient (3 hours per week against 70,000 images = decade+ to equilibrium) -> licensing failure occurred during transient despite correct end-state design -> differentiate transient interventions (fast, partial, stabilizing—e.g., flag 200 highest-commercial images for rights confirmation) from steady-state interventions, and resist optimizing only for end state
- Query failure was a sharp transition, not gradual degradation, triggered by external parameter (time-critical licensing confirmation) crossing a threshold -> the same archive and process behave qualitatively differently depending on commercial urgency and rights uncertainty -> identify minimum change to system structure that moves the bifurcation threshold, e.g., pre-confirm and pre-tag 50 images most likely to appear in licensing requests, to prevent the specific failure mode without solving the general problem
