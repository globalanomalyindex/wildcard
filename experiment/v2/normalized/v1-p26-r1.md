## angles
- Isolated colonies (each translator's glossary touching no other) produce divergence (three German names for the same loyalty program) -> terminology diffuses passively across a network only if colonies can touch -> establish a single, authoritative terminology file that every translator reads from, the minimum shared surface all five languages could touch
- English source copy changes weekly without notice, translators have no buffer -> the mat's retention structure decouples internal state from external noise -> create an authoritative, versioned English glossary that updates on a deliberate cadence (weekly, per release) distinct from live marketing pages; translators sync to that versioned buffer, not directly to live pages
- Change notification currently requires maintaining a routing table of which translator touched which page in which language, expensive under weekly edits -> broadcast plus environmental filter scales without a routing table -> emit a cheap signal (even just "last modified" timestamp in shared spreadsheet) whenever an English page changes; let translators or their tools passively filter for pages in their language
